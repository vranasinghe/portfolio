## 🛠️ Built With

This portfolio is built with modern web technologies to ensure a fast, interactive, and beautiful user experience:

- **Framework:** [Next.js 15](https://nextjs.org/) (React 19)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics:** [Three.js](https://threejs.org/), [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber), & [Spline](https://spline.design/)
- **Typography & Icons:** `react-simple-typewriter`, `lucide-react`, and `react-icons`
- **Language:** TypeScript

## ✨ Features

- **Interactive 3D Elements:** Immersive 3D components built with Spline and React Three Fiber.
- **Dynamic Animations:** Smooth page transitions, scroll animations, and typing effects powered by Framer Motion and custom libraries.
- **Responsive Design:** Fully responsive layout tailored for all screen sizes.
- **Dark/Light Mode:** Seamless theme switching for better accessibility.
- **Project & Experience Showcase:** Comprehensive sections highlighting my work in Data Science, Full-Stack Development, Mobile Apps, and AI.

## 💻 Getting Started

To run this project locally, follow these steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vranasinghe/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) (or whichever port Next.js is running on) to view the portfolio.

## 📂 Project Structure

- `app/` - Next.js App Router configuration, layouts, and global styles.
- `components/` - Reusable React components (UI components, 3D canvases, and main sections like Hero, About, Projects).
- `data/` - Centralized content configuration (`portfolio.ts`) to easily update personal information, skills, and projects without touching the UI code.
- `public/` - Static assets, images, certificates, and fonts.
